Hello Markdown

发表于 2017-08-18 | 分类于 教程

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。markdown相比于HTML来言更加轻量，熟练掌握其中基本语法规则就可以轻松写出不错的文章。

认识 Markdown

简介

markdown是一种简洁的标记语言，没有html一样复杂的标记。它用简洁的语法代替了排版，不像word等文字处理工具一样有很多操作，这使得它在文字编排方面实用性大大增加，使我们能够专注于写作，而不是文字排版。

Markdown 基本语法

标题

标题是每篇文章常用的格式。只需在文字前面加#即可。

# 一级标题

## 二级标题

### 三级标题

以此类推，Markdown内有六个级别标题。

注意：在 # 标记与文字之间需加一个空格，否则设置标题失效。Markdown大多数标记与正文间都需加一个空格才可以生效。

引用

在文本前加 > 和一个空格

列表

在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1. 2. 3. 注意符号要和文字之间加上一个字符的空格。

  1. 这是
  2. 有序
  3. 标题
  • 这是
  • 无序
  • 标题

粗体和斜体

粗体在需要加粗的文本前后加上 ** ,斜体在需要斜体的文本前后加上 * 在标记与文字间可以不加空格。
