VPS + ss 搭建VPN

  • undefined
  • 2017-09-22
  • 搭建VPN

Why can’t we be more free？

搭建VPN，说白了就是用来翻墙。记得小学时候似乎，谷歌还能流畅的访问，现在“没落到主页都404的小公司”
ヽ(￣▽￣)ﾉ。现在中国的墙是越来越高越来越结实了，不少的翻墙软件已经GG，我之前续了一年的green vpn说停就停，还好把钱返还了。所以新的风暴已经出现，怎么能够停滞不前(°Д°)。来搭一个自己的vpn吧。

VPS

什么是VPS

VPS,简单来说就是虚拟服务器。相比于虚拟主机来说，更贵，但自由度更高，你想干啥干啥。

购买VPS

常见的VPS提供商有Linode/AWS/Bandwagon/DigitalOcean/Vultr，我翻过很多博客见他们都是用的Vultr，毕竟是大牌子，有保障。其实还有很多vps供应商，甚至可以去某宝上淘一个。

学长推荐的这个站点赵容部落,有很多小众厂商，最后我对比了几家后，选了一个40多一个月的VPS，由VPS分享组织提供,配置是双核，内存4G，30GB硬盘，对外网速是3MBPS，不限流量。（我就看好不限流量了，实测3兆网速看油管的720p视频无压力）

配置运行shadowsocks

安装node.js

打开你的vps操作界面，（在配置安装vps时选windows的操作系统吧，毕竟熟悉，好操作（大佬请忽略本条））（一般有VNC操作平台，可以远程控制vps）。选择一个浏览器，访问https://nodejs.org/en/，下载并安装node.js（一般默认安装即可，安装路径可以找一个方便的路径，之后要用到）.

安装shadowsocks

cmd打开命令执行窗口，输入npm install -g shadowsocks安装ss，（这是全局安装）。

配置ss

找到C:\用户\Administrator\AppData\Roaming\npm\node_module\shadowsocks(找不到就搜索一下=。=)，用编辑器（记事本）打开config.json，将其中的server后的值改为你的vps的ip地址，（你买完vps后查看配置信息里会有的）,password后的值自己设置一个即可，其他的默认就行。

运行ss

cmd打开命令行执行窗口，输入ssserver开启shadowsocks，当你看到类似红框中的3条信息就成了，下里的是安卓端请求信息，访问了谷歌的请求是安卓端ss验证vpn是否可用。

当你访问其他墙外网站时，访问请求都会在这显示。

配置运行windows端ss

配置windows的ss

https://sourceforge.net/projects/shadowsocksgui/?source=typ_redirect下载windows版的ss（下不来就网上找），直接解压，打开里面的.exe文件。

服务器ip就是vps的ip，端口是你在配置config.json里的端口，默认是8388，密码是你设置的密码（注意不是vps的密码也不是你开机密码而是在config里设置的密码），备注起个名。

开启ss

然后在右下角就会生成一个小飞机图标，右键打开列表,服务器里选择备注名的，顺便把启用系统代理选上。

你就可上谷歌了。

配置运行安卓端的ss

下载一个shadowsocks，配置的内容和windows端的内容一样。

你可以上谷歌了。
